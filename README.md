# Rubicorn Technologies - Lodge Booking Website Proposal

An interactive, premium sales proposal and quotation web application designed for **Rubicorn Technologies Pvt Ltd** to present their Premium AI-Powered Lodge Booking Website package to clients.

## Key Features
- **Vertical Breakdown of Pricing**: Transparantly details the ₹30,000 Setup & Integrations charges across Cloud Hosting, Security Suite, Domain/SSL, AI Brand Protection, Email Engine, and Payment Gateways alongside the fixed ₹20,000 Core Website Development cost.
- **Investment Calculator**: Toggles between Monthly, 6 Months, and Annual maintenance plans to display setup costs and recurring support fees (factoring in the 6 Months FREE promotion).
- **ROI Return Calculator**: Interactive sliders for rooms count, average rate, and occupancy boost rate to display monthly/annual revenue gains and setup payback speed.
- **E-Signature Sign-off**: Integrated signature pad (HTML5 Canvas) to sign, submit, and generate a contract sign-off receipt dynamically.
- **Print to PDF Optimization**: Hand-crafted CSS stylesheets optimized for screen-to-print conversion, generating clean, professional invoice layouts when using the "Download PDF" button.
- **Theme Toggle**: Smooth transitions between a dark theme and a high-contrast light theme.

## Technology Stack
- **Core**: Semantic HTML5, Vanilla JavaScript (ES6+), CSS3 Variables & Layouts (Grid/Flexbox).
- **Fonts**: Google Fonts (`Outfit` for headers, `Plus Jakarta Sans` for body text).
- **Icons**: Inline SVG vectors.
- **Dev Server**: Vite.

## Running Locally
1. Clone the repository and navigate into the folder:
   ```bash
   git clone https://github.com/sukumar38140/Rubicorn-quotation.git
   cd Rubicorn-quotation
   ```
2. Install the development server:
   ```bash
   npm install
   ```
3. Run the application:
   ```bash
   npm run dev
   ```
4. Open your browser and navigate to the local link (typically `http://localhost:5173/`).
